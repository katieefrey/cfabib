Classification Instructions


Basic Instructions

In all interface options, you will see a list of author/affiliation pairs, grouped by article. Select the checkbox to the left of record you want to modify and click the appropriate button.

If the record you are looking at does not have affiliation information, or is unclear, click the Bibcode link to get to the ADS record of the paper. From there, you can navigate to the full article HTML page or PDF, which you can use to check up on the author's affiliation. Please check the affiliation for each author record listed in the Bibliography tool.

See "Classification Tips" for more help with deciding how to classify author/affiliation paris


Interface Options

The Author Update page is the main record evaluation interface. Here you are presented with two options: you can look at records that haven't been updated yet, or you can look at records that have already been modified.

The All Unknowns section is where all records that haven't yet been classified, plus all records classified as "unknown" are listed.

Finally, the Unverified interface is where all the records automatically classifying by the import script can be found. Records here should be checked to verify if they belong in the CfA Bibliography or not. Verification is not necassary, but it is useful to have someone double check how the script is automatically classifying its search results.


Classification Buttons

To being, select the checkbox to the left of record you want to modify and click the appropriate button.

Author Update & Unknown Interface

Select the button that best fits the provided affiliation information, or that you find on the paper.


If the author has a CfA affiliation use this button. This button and the next button will be used most often.


If the author does NOT have a CfA affiliation, use this button. This button and the above button will be used most often.


If you can tell by the authors affiliation that they are a member of the Harvard College Observatory, use this button.


If you can tell by the authors affiliation that they are a member of the Smithsonian Astrophysical Observatory, use this button.


If the author is the Director of the CfA, or the Head Librarian of Wolbach, or if you can tell by the author's affiliation that they are a member of both the Smithsonian Astrophysical Observatory AND the Harvard College Observatory, use this button. Note, this button will be used very rarely!!


If you cannot confidently determine if the author is a member of CfA at all (such as being unable to find any affiliation information or being presented with conflicting or confusing affiliation information), use this button.


Author Update & Unknown Interface

For this interface, you just need to verify if the Status given is correct or not. The red is userd to highlight records where the script guessed the author was NOT a member of the CfA.


If the given Status on the record is correct, select the record and click this button to verify the script guessed correctly.


If the Status is incorrect or if you are uncertain, select the record and use this button. Records marked as incorrect will show up in the "Author Update" interface the next time that page is loaded.
