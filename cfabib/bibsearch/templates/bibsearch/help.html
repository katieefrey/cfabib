CfA Publications Report Help & Instructions



What Does This Site Do?

This site was designed by Katie Frey at the Wolbach Library of the Center for Astrophysics to generate publication statistics reports from the CfA Bibliography. It allows someone to submit a list of names, for example the researchers within a department, to automatically generate publication stats drawn from the CfA Bibliography. Currently it is restricted to the CfA Bibliography, but there are plans to expand it to other bibilographic groups curated within the Astrophysics Data System. Contact Katie Frey if you think you should have access.

What is the CfA Bibliography?

The CfA Bibliography is a list of articles written by members of the CfA Community, carefully curated and vetted by the Wolbach Library. New articles are added to the list at least once a month. The list is maintained in the Astrophysics Data System (ADS), and the ADS interface can be used to search through the CfA Bibliography. This application leverages the ADS search capabilities to generate publication reports.

Snapshots

Reports generated by this application are a snapshot. They represent the articles published by a given author over a given time frame that were available at the moment the report was generated. Some publishers make their articles available online early, and some make them available late. It is possible that two reports for the same authors over the same time frame, but run a month apart from each other will generate different numbers. Typically the report generated at a later date will have larger numbers, especially in regards to the citation counts. This is normal and to be expected.

Getting Set Up

Before you can start using this application, there are a few things you need to do to get set up. You need to have an account with the Astrophysics Data System, and you need to add your API Access Token to your account on this site. It is also recommended that you set up your Library Link Server with ADS so that you can access the full text of most articles. The steps below will walk you through everything you need to do.

Create an ADS Account

In order to search through the CfA Bibliography, you will need an account with the Astrophysics Data System. If you do not have one, create a new account. If you already have an account, sign in.

Set up your Library Link Server

Next, it is recommended that you set up the Library Link Server in your ADS Account so that you can access the full text of articles of journals subscribed to by Wolbach and the Harvard Library.

  1. In ADS, go to Settings under the Account menu, and then to Library Link Server.
  2. Click on the "Update" button.
  3. A drop down menu should appear; select "Harvard University Library," then click the "Apply" button.
  4. You should see a line that says "Current Link Server: Harvard University Library."
You can now access full text content via Harvard's subscriptions. If you come across an article you need to access but is unavailable via our subscriptions, please send an email to library@cfa.harvard.edu or use our request form.

ADS API Token

The most important part of your ADS set up is to get your ADS API Token.

  1. In ADS, go to Settings under the Account menu, and then to API Token.
  2. Click on the "Generate a new key" button.
  3. A long string of what looks like random numbers and letters should appear in a box. Highlight the string, right click and copy.
  4. Back here at the CfA Publication Reports application, go to "Settings" under "Account," paste that string to the ADS API Token field and click "Update Token."

Generate a New Report

Now that you are all set up, you can use this site to generate publication reports. First, you'll need a list of researcher names to search for. Each name should be on a seperate line, and written lastname, firstname. Example:
Smith, Jane
Smith, John

  1. Click on "Generate New Report" found under "Publication Rerpots" in the top menu.
  2. Paste the list of names into the text box.
  3. Write in the date range you wish to query. For example, if you are looking for all articles published by your list of researchers in 2019 you can simply write "2019" in both boxes.
  4. Click the "Search" Button to add your query to the queue.
It can take anywhere from a few seconds to several minutes to execute your query, depending on the number of authors and the date range you selected.

Reports

In order to see if your most recent query has completed, and to see your past reports, click on "Report History" underneath "Reports" from the top menu.

Understanding Report History

The Report History page lists all reports you have generated using this application. It is sorted so that your most recent report is at the top, and the oldest is at the bottom. If your most recent query has not completed yet, you will see "(pending)" for it's entries. Completed reports will display a summary of how many journals and authors were found, along with a reminder about the date range included in the report.

As mentioned above, these reports are snapshots made at the moment of the query, so the Date Created for each report is also displayed prominently in this history list.

Reading a Report

When you are looking at a specific report, the sidebar on the right will remind you of the query you executed to generate this report, such as the date range, and the author list. It also displays the date and time the report was created, which is important since the information is a snapshot generated at that date and time.

The main content of the report shows a couple of graphs.
  1. The first graph shows the number of articles (refereed and non refereed) authored by a particular person.
    • Articles in which they were the "first author" are emphasized with a darker shade. If more than one person on your list were co-authors on an article, the article will be counted once for each author in this graph and in its related tabular data.
  2. The second graph shows the total number of articles written by the authors in this query that were published in each journal. If more than one person in your list were co-authors on an article, the article is only counted once in this graph and in its related tabular data.
  3. Below the graphs is some tabular data displaying the number of refereed & non refereed articles authored by a specific person, how many of those were they the first author for, and how many citations those articles have received to date. Keep in mind that if any articles have co-authos in your query list, the article will be counted once for each co-author in the author tables, but only once total in the journal table.
    • The name of each author is a link out to ADS that dispalys the articles for that person which have been gathered by this report.

Refereed Summaries

Under the "Reports" menu item is an option for "Refereed Summaries." This report will collect the total number of refereed articles and the sum of their refereed citations for each year of a provided range. When viewing a summary, you can click through to the full ADS results as well as export the tabluar data.

Exporting Graphs

When you mouse over a graph, at the top you should see a series of icons appear. The leftmost icon looks like a camera. Click that icon to download a .png image of the graph.

Exporting Spreadsheets

To export a spreadsheet of the data found in a report, click the blue "Export Spreadsheet" button found under the Tabular Data section. A CSV file will be downloaded, which you can open in Excel to save as a .xlsx file.
