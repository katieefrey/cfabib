About


This Publication Reports web application is a project from Wolbach Library.

What Does This Site Do?

This site was designed to generate publication statistics reports from the CfA Bibliography. It allows someone to submit a list of names, for example the researchers within a department, to automatically generate publication stats drawn from the CfA Bibliography. It was developed to help departmental administrators produce standard reports. There are plans to expand this tool in the future so that anyone who has a managed bibliography in the Astrophysics Data System could use the tool.

For more details about how to use this site, see the Help page.
